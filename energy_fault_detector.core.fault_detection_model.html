energy_fault_detector.core.fault_detection_model

Template for anomaly detection models

class FaultDetectionModel(config=None, model_directory='models', model_subdir=None)

Bases: ABC

Template for fault detection models. Train and predicts should be implemented.

Parameters:

  • config (Optional[Config]) – a Config object with anomaly detection configuration.

  • model_directory (str) – directory with models.

  • model_subdir (Optional[Any]) –

    if given, the subdirectory of the model objects. If None, the current datetime (formatted yyyyMMdd_HHmmSS) is used. Complete path to models:

    • train method: model_directory/model_subdir/asset_id

    • tune method: model_directory/model_subdir/asset_id_tune_method
